# Tag操作


## 循環切換 Tag

| 按鍵組合 | 功能 |
| --- | --- |
| `Alt + a` | 切換到 上一個 Tag |
| `Alt + s` | 切換到 下一個 Tag |


## 切換 Tag

| 按鍵組合 | 功能 |
| --- | --- |
| `Alt + 1` | 切換到 Tag 1 |
| `Alt + 2` | 切換到 Tag 2 |
| `Alt + 3` | 切換到 Tag 3 |
| `Alt + 4` | 切換到 Tag 4 |
| `Alt + 5` | 切換到 Tag 5 |


## 將 Window 移動到 指定 Tag

| 按鍵組合 | 功能 |
| --- | --- |
| `Win + 1` | 將目前聚焦的 Window 移動到 Tag 1 |
| `Win + 2` | 將目前聚焦的 Window 移動到 Tag 2 |
| `Win + 3` | 將目前聚焦的 Window 移動到 Tag 3 |
| `Win + 4` | 將目前聚焦的 Window 移動到 Tag 4 |
| `Win + 5` | 將目前聚焦的 Window 移動到 Tag 5 |
