# 開啟應用程式 (Terminal)

| 按鍵組合 | 功能 |
| --- | --- |
| `Alt + Enter` | 開啟 Terminal (sakura) |
| `Alt + Shift + a` | 開啟 Terminal (sakura) |
| `Alt + Ctrl + a` | 開啟 Terminal (xfce4-terminal) |
| `Alt + Shift + t` | 開啟 Terminal (xterm) |
| `Alt + Ctrl + t` | 開啟 Terminal (urxvt) |
