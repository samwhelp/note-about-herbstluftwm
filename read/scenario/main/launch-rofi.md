# 開啟應用程式 (Rofi)

| 按鍵組合 | 功能 |
| --- | --- |
| `Alt + Shift + r` | 開啟 Rofi (rofi -show run) |
| `Alt + Shift + w` | 開啟 Rofi (rofi -show window -show-icons) |
| `Alt + Shift + d` | 開啟 Rofi (rofi -show drun -show-icons) |
