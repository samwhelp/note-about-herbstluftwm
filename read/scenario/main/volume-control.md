# 音量控制


| 按鍵組合 | 功能 |
| --- | --- |
| `Alt + Shift + v` | 開啟音量控制面板 (mate-volume-control) |


| 按鍵組合 | 功能 |
| --- | --- |
| `Alt + m` | 切換靜音 |


| 按鍵組合 | 功能 |
| --- | --- |
| `Alt + Shitf + ,` | 降低音量 |
| `Alt + Shitf + .` | 加大音量 |
| `Alt + Ctrl + ,` | 緩慢地降低音量 |
| `Alt + Ctrl + .` | 緩慢地加大音量 |



| 按鍵組合 | 功能 |
| --- | --- |
| `XF86AudioMute` | 切換靜音 |
| `XF86AudioLowerVolume` | 降低音量 |
| `XF86AudioRaiseVolume` | 加大音量 |
