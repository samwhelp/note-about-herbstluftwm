# Tag操作


## 循環切換 Tag

| 按鍵組合 | 功能 |
| --- | --- |
| `Alt + ,` | 切換到 上一個 Tag |
| `Alt + .` | 切換到 下一個 Tag |


## 切換 Tag

| --- | --- |

| `Alt + 2` | 切換到 Tag 2 |
| `Alt + 3` | 切換到 Tag 3 |
| `Alt + 4` | 切換到 Tag 4 |
| `Alt + 5` | 切換到 Tag 5 |
| `Alt + 6` | 切換到 Tag 6 |
| `Alt + 7` | 切換到 Tag 7 |
| `Alt + 8` | 切換到 Tag 8 |
| `Alt + 9` | 切換到 Tag 9 |
