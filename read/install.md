# 安裝

## 如何安裝 Herbstluftwm

### Manjaro

在「Manjaro」環境，請參考「 manjaro 探索筆記 / [herbstluftwm](https://samwhelp.github.io/note-about-manjaro/read/adjustment/wm/herbstluftwm.html)」。

執行

``` sh
pamac install herbstluftwm
```

### Ubuntu

在「Ubuntu」環境，

執行

```
sudo apt-get install herbstluftwm
```
